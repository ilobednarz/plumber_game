## About

This is implementation of a Plumber game using Python and Pygame.

## Instalation (Linux)

First install all dependencies:

* Python >=3.5.2

```sudo apt-get install python3.5```

* Python libraries: Pygame, Pandas

```sudo pip3 install pygame pandas```


Then clone this repository:

```git clone https://github.com/ilobednarz/plumber_game.git```


#### Start game

Go to cloned repository directory:

```cd plumber_game```

Execute script:

```python3 play.py```


## Instructions

#### English version

This game lets you take the role of a plumber and place the pipes in such a way that the water can flow directly to the plant.

At the beginning we can see the start screen, where we can move to the first level or exit using the appropriate buttons. We also have an option to turn off the sound. Then the first level begins. The pieces of pipe in disorder appear. When we click on the element, it is rotated 90 degrees clockwise. The goal is to set them so that the water can flow from the tap in the upper left corner, to the lower right corner and water the plant. It should be done in as few moves as possible. We can see the moves count on the bar in the upper right corner, next to the sound on/off switch. Once the pipes are properly positioned, turn on the water by clicking on the red wheel. If they have actually been set correctly, we can move to the next level, and if not, we can try again. There are four levels available.


## Instrukcja

#### Wersja polska

Gra polega na wcieleniu się w rolę hydraulika i ustawieniu elementów rury w taki sposób, aby woda mogła swobodnie przepłynąć od kurka do kranu i podlać roślinkę.

Na początku widzimy ekran startowy, gdzie mamy możliwość rozpoczęcia gry lub wyjścia za pomocą odpowiednich przycisków. Mamy również możliwość wyłączenia dźwięku. Następnie rozpoczyna się pierwszy poziom. Naszym oczom ukazują się elementy rury znajdujące się w nieporządku. Po kliknięciu na element, zostaje on obrócony o 90 stopni zgodnie z ruchem wskazówek zegara. Celem jest ustawienie ich tak, aby woda mogła przepłynąć od kurka w lewym górnych rogu ekranu, do kranu w prawym dolnym rogu i podlać roślinę. Należy zrobić to w jak najmniejszej liczbie ruchów. Licznik ruchów znajduje się na pasku w prawym górnym rogu wraz z przyciskiem wyłączania dźwięku. Gdy rury są już poprawnie ustawione, należy odkręcić wodę klikając na czerwone koło. Jeśli rzeczywiście zostały ustawione poprawnie, przejdziemy do następnego poziomu, a jeśli nie, będziemy mieć możliwość spróbować ponownie. Dostępne są cztery plansze o różnym stopniu trudności.
